# 차트 분석 에이전트 가이드

## 개요

12개 분석 기법(기술 6 + 퀀트 6)을 tool로 정의하고, LLM이 결과를 종합 판단하는 에이전트 시스템.

---

## 분석 기법 목록

### 기술적 분석 (6개)

| # | 기법 | 설명 |
|---|------|------|
| 1 | 이동평균선 배열 | 골든/데드크로스, 정배열/역배열 판단 |
| 2 | RSI 다이버전스 | 과매수/과매도 + 가격-RSI 괴리 탐지 |
| 3 | 볼린저밴드 스퀴즈 | 밴드폭 수축/확장, %B 위치 |
| 4 | MACD 모멘텀 | 시그널 크로스, 히스토그램 가속/감속 |
| 5 | ADX 추세 강도 | 추세 유무, 방향, DI 크로스 |
| 6 | 거래량 프로파일 | OBV, 매집/분산, 거래량 이상 |

### 퀀트 분석 (6개)

| # | 기법 | 설명 |
|---|------|------|
| 7 | 피보나치 되돌림 | 주요 레벨(0.382/0.5/0.618) 지지/저항 |
| 8 | 변동성 체제 | ATR 퍼센타일, 체제 분류, 연환산 변동성 |
| 9 | 평균 회귀 (Z-Score) | 다기간 Z-Score, 회귀 확률 |
| 10 | 모멘텀 순위 | 1주/1개월/3개월 가중 수익률, 가속도 |
| 11 | 지지/저항선 | 피봇포인트 + 스윙 고저점, R:R 비율 |
| 12 | 수익률 자기상관 | Hurst 지수, 추세 지속/회귀/랜덤워크 판단 |

## 사용법

### 기본 실행 (LLM 없이, 12개 기법 자동 전수 분석)

```bash
python main.py AAPL --agent
```

결과: 12개 기법별 점수(-10~+10) + 종합 스코어 + JSON + 차트 + 리포트

### LLM 에이전트 연동

```bash
# Ollama (전수 분석 후 LLM이 종합 판단)
python main.py AAPL --agent --ai ollama

# GPT-4o (function calling으로 LLM이 tool 선택)
python main.py AAPL --agent --ai gpt4o

# 에이전트 + 텔레그램 전송
python main.py AAPL --agent --ai ollama --telegram
```

### 일괄 분석

```bash
# watchlist 전체를 에이전트로 분석
python main.py --watchlist --agent --ai ollama --telegram
```

## 동작 방식

### 모드별 차이

| 모드 | 동작 |
|------|------|
| `--agent` (LLM 없음) | 12개 tool 전수 실행 → 평균 점수로 자동 판단 |
| `--agent --ai ollama` | 12개 tool 전수 실행 → 결과를 Ollama에 전달 → LLM 종합 판단 |
| `--agent --ai gpt4o` | GPT-4o가 function calling으로 필요한 tool 선택 → 결과 수신 → 최종 판단 |

### 점수 체계

각 tool은 -10 ~ +10 사이의 점수를 반환한다.

| 종합 점수 | 신호 |
|-----------|------|
| +2 초과 | BUY |
| -2 ~ +2 | HOLD |
| -2 미만 | SELL |

## 출력 파일

`--agent` 실행 시 `output/` 디렉토리에 3개 파일이 생성된다.

| 파일 | 내용 |
|------|------|
| `{TICKER}_agent_YYYYMMDD.json` | 12개 tool 전체 결과 + 종합 스코어 + LLM 판단 |
| `{TICKER}_agent_YYYYMMDD.txt` | 텍스트 리포트 |
| `{TICKER}_agent_analysis.png` | 가격 차트 + 12개 tool 스코어 바 차트 |

## 기존 분석과의 관계

`--agent`는 기존 분석(step 1~7)에 추가되는 것이다. 기존 분석은 그대로 실행되고, step 5와 6 사이에 에이전트 분석이 삽입된다.

```
[1] 데이터 수집
[2] 기술 지표 계산
[3] 리스크 분석
[4] 차트 생성
[5] AI 분석 (--ai)
[5.5] 에이전트 분석 (--agent)  ← 추가
[6] 리포트 저장
[7] 텔레그램 전송
```

`--ai`와 `--agent`를 동시에 쓰면 기존 AI 분석과 에이전트 분석이 모두 실행된다.
