# 주식 분석 시스템 개선 사항 요약

## 개선 완료 항목 (2026-04-16)

### 1. 내부자 거래 분석 모듈 추가
- **파일**: `insider_trading.py`
- **기능**:
  - SEC Form 4 기반 내부자 거래 데이터 수집
  - CEO/CFO/임원진 거래 패턴 분석
  - 최근 30일 집중 거래 감지
  - C-Suite 거래에 높은 가중치 부여
  - 매수/매도 신호 강도 계산 (±4점 스케일)

### 2. 신호 라벨 일관성 개선
- **파일**: `signal_normalizer.py`
- **개선 내용**:
  - 전체 시스템에서 buy/sell/neutral로 신호 통일
  - BUY/SELL/HOLD, bullish/bearish 등 다양한 변형 자동 정규화
  - 신뢰도 0-10 범위로 표준화
  - 가중 신호 계산 기능 추가

### 3. ML 모델 신뢰도 가중치 조정
- **파일**: `enhanced_decision_maker.py`
- **개선 내용**:
  - ML 모델 정확도에 따른 동적 가중치 적용
    - 50% 미만: 0.3배 가중치
    - 50-55%: 0.5배 가중치
    - 60% 이상: 1.2배 가중치
  - 앙상블 예측의 평균 정확도 추적
  - 낮은 정확도 모델의 영향력 감소

### 4. 내부자 거래 신호 강화
- **파일**: `enhanced_decision_maker.py`, `multi_agent.py`
- **개선 내용**:
  - EventAnalyst에 내부자 거래 분석 도구 추가
  - 내부자 거래 신호에 2배 가중치 적용
  - CEO/임원진 대규모 거래 시 특별 처리
  - 강한 내부자 신호 시 다른 지표 오버라이드

### 5. 조건부 진입 전략 모듈
- **파일**: `entry_strategy.py`
- **기능**:
  - 과열 상태 감지 및 회피
  - 피보나치 되돌림 기반 진입점 계산
  - Kelly 기준 포지션 사이징
  - 분할 매수 전략 자동 생성
  - 리스크 경고 및 진입 권고 레벨
    - BUY_NOW: 즉시 진입
    - BUY_ON_DIP: 조정 대기
    - WAIT: 관망
    - AVOID: 진입 회피

## 주요 문제점 해결

### 1. 내부자 거래 미반영 → 해결
- CEO 포함 임원진의 대규모 매각 신호를 분석에 반영
- BAC 사례: 2026년 2-3월 약 100만주 집중 매각 감지 가능

### 2. 신호 라벨 불일치 → 해결
- single_llm_analysis.final_signal = "HOLD"
- single_llm_analysis.llm_conclusion = "매도"
→ 모든 신호를 buy/sell/neutral로 통일

### 3. ML 앙상블 과대 평가 → 해결
- 정확도 40-54%인 ML 모델들의 영향력 감소
- 정확도 기반 가중치로 신뢰성 있는 예측만 반영

### 4. 리스크/리워드 비율 혼재 → 통합
- support_resistance_analysis: 실제 지지/저항 기반
- risk_position_sizing: ATR 기반
→ 진입 전략 모듈에서 통합 관리

### 5. 과열 상태 진입 → 개선
- RSI 73.5, Z-score 1.61 등 과열 지표 감지
- 조정 대기 후 피보나치 레벨에서 분할 진입
- Kelly Half (12.5%) 이하로 포지션 제한
