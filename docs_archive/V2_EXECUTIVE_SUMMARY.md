# Stock AI Agent V2.0 - 경영진 요약

**한 줄 요약:** AI 주식 분석 시스템을 외부 AI와 연동하고 멀티에이전트로 정확도를 높이는 4주 프로젝트

---

## 🎯 핵심 메시지

### 왜 V2.0인가?
- **현재 한계:** 단일 AI의 편향된 판단, 수동 포트폴리오 관리, 외부 AI 연동 불가
- **V2.0 솔루션:** 6명의 전문가 AI 협업, 자동 리밸런싱, Claude/ChatGPT 직접 호출

### 3대 핵심 기능

#### 1. MCP 서버 (1주 개발)
```
Claude: "NVDA 분석해줘"
   ↓
MCP 서버가 16개 도구 실행
   ↓
"매수 신호, 신뢰도 85%, RSI 과매도 + MACD 골든크로스"
```

#### 2. 멀티에이전트 (2주 개발)
```
기술 분석가: "차트상 매수"
퀀트 분석가: "통계적 매수"
리스크 관리자: "중립, 변동성 주의"
ML 전문가: "상승 확률 73%"
이벤트 분석가: "실적 호조"
   ↓
의사결정자: "종합 매수, 단 변동성 리스크 관리 필요"
```

#### 3. 자동 리밸런싱 (1주 개발)
```
매주 월요일 09:30
   ↓
목표 비중 vs 현재 비중 체크
   ↓
5% 이상 차이 시 자동 조정
   ↓
거래비용 0.1% 반영
```
