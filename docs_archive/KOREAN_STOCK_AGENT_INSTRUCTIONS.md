# 한국 주식 분석을 위한 에이전트 지시문
## Korean Stock Analysis - Agent Instructions

### 📋 개요
본 지시문은 AI 에이전트(ollama 등)가 한국 주식 시장을 분석할 때 참고해야 할 특별 지침입니다.

---

## 1. 🇰🇷 한국 시장 특성

### 시장 구조
- **KOSPI (한국종합주가지수)**: 대형주 중심, 시가총액 기준
- **KOSDAQ (코스닥)**: 중소형 성장주, 기술주 중심
- **거래시간**: 09:00 - 15:30 KST (점심시간 없음)
- **통화**: KRW (원화)
- **가격 단위**: 주가에 따라 호가 단위 상이
  - 1,000원 미만: 1원
  - 1,000~5,000원: 5원
  - 5,000~10,000원: 10원
  - 10,000~50,000원: 50원
  - 50,000원 이상: 100원

### 가격 제한폭
- **상한가/하한가**: 전일 종가 대비 ±30%
- **서킷 브레이커**: KOSPI/KOSDAQ 지수 8%, 15%, 20% 하락 시 발동

## 3. 🎯 Multi-Agent 분석 가이드

### Technical Analyst (기술적 분석가)
```
한국 주식 기술적 분석 시 추가 고려사항:

1. 거래량 분석
   - 개인: 거래량 1위
   - 외국인/기관: 큰 손 매매
   → 거래량 + 외국인 순매수 = 강한 신호

2. 차트 패턴
   - 상한가/하한가 근접 여부
   - ±30% 제한폭 영향
   - 갭 상승/하락 빈도

3. 이동평균선
   - 5일선, 20일선, 60일선, 120일선
   - 한국 투자자들의 선호 기준선

4. 매물대 분석
   - 심리적 저항선 (10,000원, 50,000원, 100,000원)
   - 원화 단위 특성 반영
```

### Quant Analyst (퀀트 분석가)
```
한국 주식 퀀트 지표:

1. 상대적 모멘텀
   - KOSPI/KOSDAQ 지수 대비 강도
   - 동일 섹터 내 상대 수익률

2. 외국인 지분율
   - 30% 이상: 외국인 선호주
   - 증가/감소 추세 분석

3. 유동성 지표
   - 거래대금 상위 여부
   - 호가 스프레드
   - 시가총액 대비 거래량 비율

4. 재무 배수
   - PER (한국 평균: 10~15배)
   - PBR (한국 평균: 0.8~1.2배)
   - ROE, 부채비율
```

### Risk Manager (리스크 관리자)
```
한국 주식 리스크 요인:

1. 시장 리스크
   - 코리아 디스카운트 (저평가 현상)
   - 북한 리스크 (지정학적)
   - 중국 경기 의존도

2. 개별 종목 리스크
   - 대주주/총수 리스크
   - 지배구조 문제
   - 정부 규제 (공정거래, 산업 정책)

3. 환율 영향
   - 수출주: 원화 약세 = 실적 개선
   - 수입주: 원화 강세 = 비용 절감
   - USD/KRW 환율 모니터링 필수

4. 외국인 수급
   - 외국인 순매도 지속 시 추가 하락 압력
   - 외환 시장 연동성
```

### Event Analyst (이벤트 분석가)
```
한국 시장 이벤트:

1. 공시
   - DART (전자공시시스템) 체크
   - 실적 발표 (분기/반기/연간)
   - 주요 경영 사항 (유상증자, M&A, 배당)

2. 뉴스
   - 한국어 뉴스 감성 분석
   - 정부 정책 발표
   - 산업 동향 (반도체, 배터리 등)

3. 시즌성
   - 배당락: 12월 말 (결산월)
   - 실적 시즌: 1월, 4월, 7월, 10월
   - 주주총회: 3월

4. 글로벌 연동
   - 미국 증시 (나스닥, S&P500) 전날 동향
   - 중국 증시 (상해종합) 영향
   - 일본 닛케이 연동성
```

### ML Specialist (머신러닝 전문가)
```
한국 주식 ML 모델 특징:

1. Feature Engineering
   - 외국인 순매수 누적 (5일, 10일, 20일)
   - 기관 순매수 누적
   - 프로그램 매매 비중
   - 테마 모멘텀 지수
   - KOSPI/KOSDAQ 스프레드

2. 시계열 특성
   - 미국 증시 영향 (전일 S&P500, NASDAQ)
   - 원/달러 환율
   - 유가, 금리

3. 모델 검증
   - 한국 시장 고유 변동성 반영
   - 외부 쇼크 대응력 (북한 리스크, 환율 급변)
```

### Decision Maker (최종 의사결정자)
```
한국 주식 종합 판단 시 가중치:

1. 외국인/기관 동향: 30%
   → 큰 손 매매가 방향성 결정

2. 기술적 지표: 25%
   → 한국 투자자들의 차트 의존도 높음

3. 재무/밸류에이션: 20%
   → PER, PBR, ROE

4. 테마/섹터 강도: 15%
   → 테마주 급등/급락 빈번

5. 뉴스/이벤트: 10%
   → 공시, 정책 발표 영향

[최종 신호 산출]
- BUY: 외국인/기관 순매수 + 기술적 상승 + 테마 강세
- SELL: 외국인/기관 순매도 + 기술적 하락 + 악재
- NEUTRAL: 혼조, 방향성 불명확
```
