# Stock AI Agent Design System Revision Plan ## 1. 개요 현재 텍스트 중심의 디자인 가이드를 시각적인 'Brand & UI Kit' 형태로 재구성하여 가독성과 활용성을 높입니다. ## 2. 시각화 전략 - **Color Swatches**: Hex 코드만 나열하는 대신, 실제 색상이 적용된 칩과 용도를 병기하여 즉각적인 인지 유도. - **Typography Scale**: 폰트 종류, 굵기, 크기를 실제 예시 문장과 함께 배치하여 계층 구조 명확화. - **Component Showcase**: CSS/HTML 스니펫과 함께 실제 렌더링된 컴포넌트 샘플을 나열. - **Interactive Elements**: Hover 상태, 버튼 액션 등 인터랙티브 요소의 변화를 시각적으로 기술. ## 3. 주요 개선 항목 - **Foundations**: 다크 테마의 깊이감(Elevation) 표현을 위한 레이어별 배경색 가이드. - **Semantic System**: Buy/Sell/Hold 시그널 컬러의 일관된 적용 규칙 (배지, 차트, 텍스트). - **Chart Guidelines**: Plotly 차트와 UI 간의 시각적 일치감을 위한 데이터 시각화 표준화. ## 4. 향후 단계 1. **Design System 구축**: 위 명세서를 바탕으로 Stitch 내 Design System 토큰 생성. 2. **Component Library 생성**: 주요 컴포넌트(Summary Card, Ticker Bar 등) 개별 디자인. 3. **Template Screen 제작**: 가이드가 적용된 실제 대시보드 화면 구현.
