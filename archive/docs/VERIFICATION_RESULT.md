1. **완전 일치** (100%): 정확한 회사명/티커
2. **별칭 일치** (99%): 네이버→NAVER, 포스코→POSCO홀딩스
3. **StartsWith** (95%): 삼성에피스→삼성에피스홀딩스
4. **Contains** (85%): 부분 포함
